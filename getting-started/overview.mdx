## What CyBox Does

CyBox is a **Virtual Security Team for the AI Era**. It sits between your AI coding tools and your production environment, giving you continuous visibility and control over what gets built and deployed.

### The Problem

AI coding assistants (Copilot, Cursor, Claude, v0, Lovable, Bolt) generate code fast — but they also introduce vulnerabilities fast. Traditional scanners were not built for this pace or this pattern.

### The CyBox Approach

CyBox operates across three layers:

<Steps>
  <Step title="Discover">
    Connect your GitHub repositories and deployed web assets. CyBox automatically inventories everything — code, dependencies, secrets, infrastructure, and live URLs.
  </Step>
  <Step title="Detect">
    Every push and PR is scanned across six scanner types: SAST, SCA, Secrets, IaC, License, and DAST. Findings are deduplicated, prioritized, and mapped to the diff that introduced them.
  </Step>
  <Step title="Act">
    CyBox generates fix PRs via AutoFix, enforces security gates via PR Firewall, and surfaces live vulnerabilities via DAST — so your team can act, not just review reports.
  </Step>
</Steps>

## Who CyBox Is For

- **Engineering teams** building with AI coding tools who need automated security coverage without slowing down
- **Security leads** who need visibility into AI-generated code risk across the org
- **CTOs and founders** who need to ship fast without introducing silent risk
