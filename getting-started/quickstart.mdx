# Quickstart

This guide walks you through the fastest path to value.

## Before you begin

- A CyBox account at [app.cybox.ai](https://app.cybox.ai)
- Admin access to the GitHub organization or repositories you want to connect

## 1. Install the GitHub App

1. Open **Integrations → GitHub** in CyBox
2. Click **Install GitHub App**
3. Choose the repositories you want CyBox to access
4. Complete the GitHub authorization flow

<Note>
CyBox needs GitHub access for repository scanning, PR Firewall checks, and creating eligible AutoFix pull requests.
</Note>

## 2. Add your first repository

1. Go to **Assets**
2. Click **Add Asset**
3. Select **Repository**
4. Pick the repository from your connected GitHub installation
5. Save

## 3. Run the first scan

CyBox may start a scan automatically after the repository is connected. You can also trigger one manually from the asset page with **Scan Now**.

## 4. Review results

After the scan completes:

1. Open **Findings**
2. Filter to **Critical** and **High** first
3. Open a finding to review severity, evidence, remediation guidance, and AutoFix availability

## 5. Turn on merge-time control

Once baseline findings are understood, enable [PR Firewall](/features/pr-firewall) so new risky changes can be reviewed in pull requests before merge.
