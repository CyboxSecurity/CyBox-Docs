# What is CyBox?

CyBox helps teams detect risk, move faster on remediation, and stop dangerous changes before they reach production.

## The simple answer

CyBox is a **security control layer for AI-generated code and fast-moving software exposure**.

It is built for teams shipping with AI tools, modern deployment platforms, and rapid pull request cycles.

## Why teams need it

AI has changed how software gets built:

- More code is produced faster
- More changes reach pull requests with less review time
- More apps and preview environments go live outside formal security processes

That speed is valuable, but it also creates a control problem.
