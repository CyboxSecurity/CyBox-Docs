# Connect a repository

## Step 1
Open **Integrations → GitHub** and install the CyBox GitHub App.

## Step 2
Choose the repositories or organization scope you want to grant access to.

## Step 3
Return to **Assets** in CyBox and add the repository as a new asset.

## Step 4
Run a scan or wait for the initial scan to begin.

## What happens next

Once connected, CyBox can:

- Scan the repository
- Track findings over time
- Evaluate pull requests with PR Firewall
- Create supported AutoFix pull requests

<Tip>
Start with one important repository first. It makes rollout cleaner and easier to explain internally.
</Tip>
