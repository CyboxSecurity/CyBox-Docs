# Add a domain or website

## Add the asset

1. Go to **Assets**
2. Click **Add Asset**
3. Choose **Website** or **URL**
4. Enter the public address you want to monitor
5. Save

## After the asset is added

CyBox can use that asset for:

- Live exposure validation
- Posture review
- Ongoing monitoring

## Best first targets

- Main production website
- Customer portal
- Public staging environment
- Demo app exposed to the internet
