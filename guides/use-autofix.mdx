# Use AutoFix

## Generate the fix

1. Open a supported finding
2. Click **AutoFix**
3. Wait for CyBox to prepare the pull request

## Review the PR

Check:

- The code change itself
- The commit and PR description
- Whether the remediation matches the issue
- Whether tests or validation are needed before merge

## Merge or edit

If the change looks good, merge it. If needed, edit the proposed code before merge.

<Warning>
AutoFix accelerates remediation, but it does not replace engineering review.
</Warning>
