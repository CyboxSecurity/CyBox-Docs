# Domain monitoring

Domain monitoring helps CyBox track the public-facing side of your environment.

## What it is for

Use it to bring in:

- Main websites
- Customer-facing apps
- Preview environments
- Deployment-linked web assets

## What CyBox can do next

Once a domain or website is connected, CyBox can:

- Track the asset in inventory
- Review posture at the asset level
- Run live exposure validation
- Keep the asset visible alongside repository risk

## Suggested first rollout

Start with the most business-critical public asset, then expand to previews and lower-governance environments.
