# GitHub integration

GitHub is the main integration for repository-based CyBox workflows.

## What it enables

- Repository scanning
- Pull request evaluation with PR Firewall
- Supported AutoFix pull request creation
- Shared visibility between security and engineering

## Setup flow

1. Open **Integrations → GitHub**
2. Install the CyBox GitHub App
3. Choose repository scope
4. Return to CyBox and add repositories as assets

## Best practice

Start with a limited repository set, confirm the workflow with one engineering team, then expand.
